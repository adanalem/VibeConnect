# 🚀 VibeConnect - 3D Social Dashboard

A modern, high-performance micro-blogging application with a focus on **3D Glassmorphism** and **User Experience**. Built with a focus on "Warming" aesthetics, this dashboard allows users to share thoughts and images in a beautiful, interactive environment.

## ✨ Key Features

- **🖼️ Image Posting:** Share your moments with support for photo uploads.
- **🎨 6 Dynamic Themes:** Switch between Dark, Light, Blue, Brown, Sunset, and Forest modes.
- **🧊 3D UI Design:** Premium glassmorphism effects with depth and hover animations.
- **🔐 Secure Auth:** Full authentication system powered by **Supabase**.
- **❤️ Permanent Likes:** Interactive like system that stays saved even after refresh.
- **📱 Responsive Layout:** Fully optimized for mobile, tablet, and desktop views.
- **🛠️ Full CRUD:** Users can Create, Read, Update, and Delete their posts with confirmation alerts.

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS (Design), Vanilla JavaScript (Logic)
- **Backend/Database:** Supabase (PostgreSQL)
- **Animations:** CSS3 Transitions & 3D Transforms
- **Alerts:** SweetAlert2 for professional notifications
